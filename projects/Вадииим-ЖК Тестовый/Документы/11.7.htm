Основные изменения и дополнения версии NormCAD 11.7

В основной программе:

	- добавлена вставка в отчет таблиц в текстовом формате (без использования программы Excel):

		

	- добавлены переключатели в окне Параметры (на кладке Формат):

		- вставлять таблицы как тест
		- моноширинный шрифт

		

	- добавлен выбор шрифтов в окне Параметры (на кладке Шрифт):

		- Текстовые таблицы
		- Текстовые таблицы (моноширинный шрифт)

		

	- добавлен раскрывающийся список последних файлов у кнопки Открыть

	- обновлена и расширена библиотека API NormCAD

	- скорректированы пути расположения файлов

	- добавлен упрощенный вариант программы NormCAD - программа NormCAD LT:

		

		Особенности и преимущества программы NormCAD LT:

			-  упрощенный однооконный интерфейс

			-  ускоренный запуск программы

			-  поддержка основных (наиболее часто используемых) возможностей программы NormCAD:

				- создание документов для расчетов на основе заданий
				- открытие документов, созданных в NormCAD
				- сохранение документов в формате NormCAD
				- выполнение расчетов и подготовка отчетов
				- вывод отчетов на печать
				- показ значения основного результата расчета (в строке состояния)
				- переход к значению основного результата расчета в отчете (по щелчку в строке состояния)
				- редактирование данных по щелчку в описании данных тексте отчета
				- передача отчета в Word с оформлением рамкой и штампом
				- значок для открытия примеров (в левой части окна) 
				- значки для открытия окон по группам данных (в левой части окна) 
				- таблица данных и список условий расчета (в правой части окна программы)
				- показ результата выделенной части фрагмента формулы в отчете
				- показ примечаний и схем (в верхней части окна программы)
				- редактирование параметров расчета (кнопка Параметры)
				- редактирование справочников (раскрывающийся список кнопки Параметры)

			- значок программы NormCAD LT на рабочем столе:
